### Calculando Fatorial

```python
# Calculando o fatorial de um número usando um loop for
numero = 5
fatorial = 1

for i in range(1, numero + 1):
    fatorial = fatorial * i

print("Fatorial de", numero, "é", fatorial)
```

Neste exemplo, o loop `for` é usado para calcular o fatorial de um número (`5!` neste caso).
